## Usage

**For Figma:**
- Copy URLs directly into image placeholders
- Images load from GitHub CDN (fast, reliable)

**For Web Development:**
- Use these URLs in `<img src="">` tags
- Or as `background-image` in CSS
- GitHub serves with proper headers (caching, CORS)

**For Testing:**
- Click any URL to view the image directly
- All images return with `content-type: image/jpeg`
